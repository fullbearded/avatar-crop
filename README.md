# Avatar Crop

## Dependencies

* Mongodb
* ImageMagick

CentOS:
<pre>
yum install -y ImageMagick
</pre>

MongoDB install 
reference: http://blog.58share.com/?p=192

Mac OS:
<pre>
brew install image_magick
brew install mongodb
</pre>

## Usage

<pre>
git clone https://github.com/huhongda/avatar-crop.git

cd avatar-crop

bundle install

rails s 
</pre>
